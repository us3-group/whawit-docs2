## What the WHAWIT API does

The WHAWIT API lets you automate and extend the platform so it fits cleanly into your existing tooling.

- Syncing incident and insight data into ticketing systems.
- Building custom internal dashboards on top of WHAWITâ€™s incident understanding.

- Exporting recommendations and code-level findings into your engineering workflow.

## Authentication

WHAWIT uses Bearer tokens for authentication.

```http
Authorization: Bearer YOUR_API_TOKEN
```

Tokens should be kept secret and rotated regularly.
Use environment variables or your secrets manager rather than hard-coding tokens in source control.
